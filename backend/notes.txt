=== Instructions for database table creation ===
- You need to add rights for creating in MariaDB

- From root write with your username to check rights
SHOW GRANTS FOR 'username'@'localhost';

- To give all rights
GRANT ALL PRIVILEGES ON flight_game.* TO 'username'@'localhost';

- To save changes
FLUSH PRIVILEGES;


=== Instructions for removing previous version of "results" table from database ===
- In MariaDB go to the DB
USE flight_game;
DROP TABLE results;
